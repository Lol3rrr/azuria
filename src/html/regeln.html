Regeln

Die normalen Regeln

Ein paar allgemeine Regeln sind:
- nicht griefen
- nicht klauen
- kein arschloch sein

Note:
Diese Regeln gelten nur Teilweise während Roleplay, da bestimmte Aktionen welche sonst verboten sind im Roleplay benötigt werden.

Gesetze

Die meisten Sachen werden durch Gesetze auf dem Server geregelt.
Da sich diese konstant ändern werden sie hier nicht aufgeführt, um die aktuellen Gesetze zu wenden sie sich bitte an die Polizei auf dem Server

Strafen

Wenn die Regeln gebrochen werden startet die Strafe bei einem 1 Tag ban. Die Dauer der Strafe nimmt mit jedem Regelbruch zu.
Die Strafen für das Brechen von Gesetzen auf dem Server wird auf dem Server geregelt.
